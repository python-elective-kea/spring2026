# Installing UV

## **1. Install UV**
UV is a fast Python package installer and resolver written in Rust. Install it using:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

**Verify installation:**
```bash
uv --version
```

--- 

## **2. Create a Virtual Environment**
UV can create and manage virtual environments:

```bash
uv venv .venv  # Create a virtual environment
source .venv/bin/activate  # Activate on Linux/Mac
# .venv\Scripts\activate  # Windows
```

--- 

## **3. Install Packages**
UV is much faster than pip for installing packages:

```bash
uv pip install requests numpy pandas
```

For development dependencies:
```bash
uv pip install -r requirements-dev.txt
```

--- 

## **4. Sync Dependencies**
UV can sync your environment with dependency files:

```bash
uv pip sync requirements.txt  # Exact sync
uv pip sync pyproject.toml     # Sync from pyproject.toml
```

--- 

## **5. Run Commands in Isolated Environments**
Run Python scripts or commands in isolated environments:

```bash
uv run python script.py
uv run pytest
```

--- 

## **Key Features**
- **Blazing Fast**: Written in Rust, significantly faster than pip
- **Virtual Environment Management**: Built-in venv creation and management
- **Dependency Resolution**: Advanced resolver for complex dependency graphs
- **Cross-Platform**: Works on Linux, Mac, and Windows
- **Python Version Management**: Can install and manage multiple Python versions

--- 

## **Additional Commands**

**Install specific Python version:**
```bash
uv python install 3.11.8
```

**List installed packages:**
```bash
uv pip list
```

**Remove virtual environment:**
```bash
rm -rf .venv
```
