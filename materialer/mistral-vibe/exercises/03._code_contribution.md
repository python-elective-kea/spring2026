# Code Contributions

Mistral-vibe projektet tager for øjeblikket ikke imod kode forslag/ændringer mm. De skriver at det eventuelt kommer senere. 
Repositoriet gør brug af en "Apache" licence, hvilket betyder at koden er "Open Source". Derfor har jeg lavet en fork af repositoriet, og i min fork må man gerne komme med kodeændringsforslag.

**Motivation**: Lær at lave et Pull Request med kode ændringer.
---

1. Lav en fork af dette repository: https://github.com/python-elective-kea/mistral-vibe 
2. Lav en ny feature, eller et bug fix.
    * Du kan evt. lade dig inspirere af disse [issues](https://github.com/mistralai/mistral-vibe/issues)
3. lav et Pull Request til https://github.com/python-elective-kea/mistral-vibe
4. hold øje med evt. kommunikation omkring dit pull request, eller om dit pull request er blevet accepteret.
